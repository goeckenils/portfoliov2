# Portfolio V2 (Experimental)

> **Tech Demo:** Exploring the bleeding edge of React 19, Next.js 16, and GSAP animations.

This repository serves as a playground and showcase for high-performance frontend techniques. It implements a modern, animation-heavy UI (inspired by Omse) using the absolute latest stack available in the React ecosystem.

## ⚡ Tech Stack

- **Framework:** [Next.js 16.1.6](https://nextjs.org/) (Canary/Latest) - Leveraging React Server Components & latest routing optimizations.
- **Library:** [React 19.2.3](https://react.dev/) - Utilizing new hooks and concurrent features.
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) - The new engine for ultra-fast styling.
- **Animation:** [GSAP 3.14](https://gsap.com/) - Complex timelines and scroll-triggered animations.
- **Type Safety:** TypeScript 5.

## 🚀 Why this stack?

I built this to test the limits of **Next.js 16** combined with heavy GSAP animations. The goal is to achieve 60fps animations while maintaining good Core Web Vitals, proving that high-end creative web design and modern framework architecture can coexist efficiently.

## 🛠️ Development

```bash
# Install dependencies
npm install

# Run dev server (Turbopack enabled)
npm run dev
```
